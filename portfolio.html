What I Hoped For and Envisioned

Html is a pretty straight-forward language; it is in English and explicit in its intention. The most confusing elements are the semantic ones; features that visually appear the same on the browser (or are not displayed at all, like structural elements) that have important intention and distinctive purpose. My hope is to concretely grasp these topics and be able to code neat, organized, purposeful code.

Further, as an artist (and kind of aspiring UX designer), aesthetic is important to me. I hope to be able to design beautiful interfaces, one's with color, images, shapes, various fonts, columns, and more. I especially want to be able to create dynamic pages, or product motion design, as Payton Meyer advised.

Thoughts on dynamic design:

  • I reached out to Payton via LinkedIn, and after speaking to him about UX design and discussing my concerns and hopes for the field, I decided I would 100% learn html and 90% pursue UX design.
  • Besides the fact that UI animation is an awesome bonus skill, I have always loved animate (and taught myself how to do very novice character and graphic animation), so I was determined to learn that too.
I just want to create. Art is creation, but above all, communication. Through learning how to communicate via html, I will be able to evolve into a strong, effective creator.
